**StarRank** is not just another rank plugin. It is a high-performance, web-synchronized authority system designed for the **StarLight** network to outperform even the biggest competitors.

## ✨ Premium Features
- 🚀 **Asynchronous Web Sync**: Powered by a unified SQLite3 database that syncs in real-time with the **StarLight Go-Backend**.
- 💎 **Dynamic Nametags**: Crystal-clear nametag formatting with level colors, rank prefixes, and smooth transitions.
- 🗡️ **Personalized Kill Messages**: Let every player leave their mark with custom, database-stored kill messages.
- 🛒 **Integrated Rank Shop**: Seamless integration for players to purchase status and authority.
- 🎨 **Unicode Aesthetic**: Full support for custom unicode symbols and premium colors.
- 🛡️ **Robust Permissions**: Built for scale, ensuring only the worthy hold the ranks.

## 🛠️ Commands
| Command | Description | Permission |
|---------|-------------|------------|
| `/setrank` | Change a player's destiny | `starrank.command.admin` |
| `/rankshop` | Browse the hall of status | `starrank.command.player` |
| `/killmessage` | Customize your signature | `starrank.command.player` |
